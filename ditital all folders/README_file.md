## 📌 Module Summaries

### Task 1 – Digital Literacy Infographic
Created a one-page visual infographic using **Canva** covering all five core topics: the meaning of digital literacy, useful digital tools for students (GitHub, LeetCode, Kaggle, Google Workspace), safe internet practices with a focus on UPI safety in India, professional online presence, and email etiquette essentials.

### Task 2 – Student Digital Portfolio
Set up professional profiles on **GitHub**, **LinkedIn**, and **Kaggle**. GitHub profile (Rishabjainwork) includes 5 public repositories showcasing Python and HTML projects. LinkedIn profile is active with institution details. Kaggle account (rishabjain1006) created for future data science practice.
